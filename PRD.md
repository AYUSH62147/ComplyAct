# Product Requirements Document & Demo Script

## Core Vision
To become the universal, auditable translation layer between unstructured real-world data and legacy enterprise systems, using AWS Nova and cryptographic governance.

## 🌟 Pinnacle Features
1. **Hyper-Sync Engine**: 2x faster orchestration between reasoning and UI automation.
2. **Nova Act Simulation**: Dynamic "Ghost Cursor" navigating legacy ERP systems.
3. **HITL Governance**: "Graceful Halt" logic for complex or handwritten data (e.g., dates).
4. **Immutability**: Cryptographic receipt generation verified via QLDB.
5. **Manageable UI**: Independent overlay dismissal for clean walkthroughs.

## 🎬 The 90-Second Demo Script
- **00s-10s (Ingestion):** Upload the handwritten invoice `handwriting_...pdf`. The scanning beam activates.
- **10s-20s (Reasoning):** Terminal shows Nova Pro extracting data in high-speed typewriter mode.
- **20s-40s (Nova Act):** The Ghost Cursor begins filling the Mock ERP. The terminal narrates every action.
- **40s-60s (The Halt):** Agent halts at the Date field. Slack modal appears with confidence scores.
- **60s-75s (Override):** User selects "Corrected Value". Agent resumes in 2x Hyper-Sync mode.
- **75s-90s (Audit):** Audit Receipt appears. User closes it to show the full terminal logs and verified form data.
